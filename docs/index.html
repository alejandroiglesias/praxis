Praxis

A modern CSS framework built with Stylus.

Currently in early development.

Praxis is open source. For reviewing code, opening issues and making pull requests, go to the GitHub project.

Grid system

Praxis ships with a powerful grid system called Lost Grid, fully mobile-first configured out of the box for your layouts.
